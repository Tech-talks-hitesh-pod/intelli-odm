# Intelli-ODM

This project leverages Python alongside the powerful Ollama llama3:8b LLM model to deliver advanced AI capabilities.

## Tech Stack

- **Python**  
  Used for core application logic and orchestration.

- **Ollama: llama3:8b**  
  Integrated as the foundational large language model. See [Ollama Documentation](https://ollama.com/) for details.

## Getting Started

### Prerequisites

- Python 3.8+
- [Ollama](https://ollama.com/) installed locally (`ollama run llama3:8b`)

### Installation

1. Clone this repository:
  
   git clone <repo_url>
   2. Set up your Python environment:
  
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   3. Start the Ollama model:
  
   ollama run llama3:8b
   4. Run your application as described in project-specific docs.

## Usage

Follow in-project documentation and scripts to utilize AI-powered capabilities.
